# ⏰ | 提醒事项  支持群聊或私聊

## __使用__
## 添加
> 添加闹钟 22:00 睡觉啦
```
user : 添加闹钟 time [note]
bot : 不重复, 设置为每日输入[Y/y], 设置自定 如周一周三输入[13]
user : [ones]

    - time 闹钟时间, 格式 H:M(24时制)  或 +30M , +1H （类似倒计时）
    - note(可选) 可以输入@ ,图片, 视频等
    - ones(可选) 输入Y或y则为设置每天, 输入数字则设置为对应星期 1234567 = y
    
```

## 删除

删除闹钟 ID
